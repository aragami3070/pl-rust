# Перестановки

## Задача

Реализуйте функцию `permutations`, которая принимает вектор и возвращает вектор всех возможных перестановок элементов исходного вектора.

## Подсказки

- Используйте рекурсию для генерации всех возможных упорядоченных последовательностей элементов.
- Рассмотрите базовый случай, когда вектор пустой.
