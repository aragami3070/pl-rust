# Комбинации

## Задача

Реализуйте функцию `combinations`, которая принимает некоторый срез и параметр `k`, и возвращает вектор всех возможных комбинаций из `k` элементов. Используйте только рекурсию и циклы.

**Примечание**: индексы элементов комбинаций должны быть в лексикографическом порядке.

## Подсказки

- Вам потребуется использовать [Vec](https://doc.rust-lang.org/std/vec/struct.Vec.html) и его срезы для решения этой задачи. Ознакомьтесь с методами `insert`, `is_empty`.

## Бонус

Постарайтесь не использовать циклы вообще. Для этого вам нужно будет изучить итераторы.
