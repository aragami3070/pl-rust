# Bogosort


## Задача

Реализуйте функцию `bogosort`, которая принимает вектор и сортирует его по возрастанию с помощью алгоритма Bogosort.

Нужно реализовать два решения: императивное и функциональное.

## Ограничения
- Запрещено использовать дополнительные конструкции `use` кроме тех, что включены в шаблон задания.

## Бонус

Постарайтесь не использовать циклы вообще. Для этого вам нужно будет изучить итераторы.
